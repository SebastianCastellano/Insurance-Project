Readme Matlab files:

Main folder:
- Group_8_SII_Project: main script of the project

Folder 'Data':
- Life_Tables_2021
- RiskFreeRate_IT_NOVA310322

Folder 'Functions':
- equity: Compute the equity value for T years using GBM and MonteCarlo method
- fwd: Compute the Forward Rates starting from the EIOPA yield curve
- capital: Compute the value of the capital to be paid in the two cases (for T years)
- liabilities: Compute the liabilities value for T years
- liabilities_MASS: Compute the liabilities value for T years in the case of MASS shock
- shock: Evaluate BOF, delta_BOF and duration of the shock
